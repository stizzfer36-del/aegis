# Phase 05 — Full Agent Society

## Mission

Expand beyond the core only when the society becomes stronger rather than noisier.

## Candidate additions

- Scout
- Prophet
- Lens
- Sense
- Curator
- Quartermaster

## Rule

No new agent exists because it sounds impressive.
Each must reduce load, add clear leverage, and fit the delegation model.

## Acceptance criteria

- Each new agent has unique jurisdiction.
- Warden can explain why it exists.
- The system becomes faster, safer, or more valuable with it.

## Proof demo

A broader multi-agent run measurably outperforms the same task executed by the core six alone.
