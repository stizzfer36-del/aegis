# Phase 02 — Unified Session

## Mission

Create the first real standing society around one shared session.

## Ships in this phase

- Warden.
- Scribe.
- Herald.
- Terminal-first session.
- Telegram bridge.
- Shared memory and trace continuity.

## Banned in this phase

- Agent proliferation.
- Lens-grade dashboards.
- Scout-grade research expansion.
- New interfaces that dilute focus.

## Acceptance criteria

- The same task can continue across terminal and Telegram.
- Warden detects clutter and invalid delegation.
- Scribe preserves intent, outcome, and lessons.
- Herald makes the session feel singular rather than channel-specific.

## Proof demo

A task starts in terminal, continues in Telegram, survives interruption, and resumes with full memory and mapped state.
