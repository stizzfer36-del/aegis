# AEGIS

**AEGIS is the sovereign digital OS of your imagination.**

It is a single, always-on society of standing agents that designs, builds, remembers, delegates, maps consequences, and executes across software, hardware, business, research, and operations from one unified session.

## Core idea

AEGIS is not a chatbot, code assistant, or automation script.
It is a Digital OS:

- One entry point.
- One unified session.
- One memory spine.
- One action-consequence map.
- One standing agent society.
- One long-term goal: turn imagination into autonomous, wealth-generating execution.

## Core six

- **Kernel** — event bus, memory client, policy gate, scheduler, router, introspection, consequence mapping.
- **Warden** — structure enforcement, delegation cleanliness, anti-clutter control.
- **Scribe** — long-term memory, compression, trace preservation, skill promotion.
- **Herald** — unified session across terminal, Telegram, and future interfaces.
- **Forge** — artifact execution across code, documents, assets, and operational outputs.
- **Loop** — autonomous backlog digestion, self-extension, and compounding output.

Everything else is secondary until these six work cleanly.

## Principles

- Local-first by default.
- Oracle Always Free + Cloudflare + phone-first operation.
- Full action-consequence mapping.
- No hidden steps on trust-critical flows.
- No destructive or wealth-affecting execution without approval.
- Plain markdown and plain contracts over magic.
- Zero clutter.

## How to use this repo

1. Read FOUNDING_CHARTER.md first.
2. Read PHASES/README.md to understand order.
3. Read the matching phase file before touching any implementation.
4. Read the folder README for your scope.
5. Use issue and PR templates in DELEGATION/.
6. Do not add code, agents, or folders not justified by the active phase.

## Definition of success

AEGIS is real when one intent can be accepted in one unified session, decomposed by the core society, fully mapped, safely executed, resumed after interruption, remembered permanently, and turned into a shipped artifact with observable value.
