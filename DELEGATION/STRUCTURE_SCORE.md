# Structure Score

## Purpose

The structure score is the anti-clutter metric.

## Positive signals

- clear ownership
- narrow responsibility
- valid contracts
- test coverage
- trace visibility
- rollback plan

## Negative signals

- overlapping scope
- undocumented behavior
- unmapped execution
- prestige complexity
- broken phase alignment

## Use

PRs, new agents, and new folders must state expected score impact.
