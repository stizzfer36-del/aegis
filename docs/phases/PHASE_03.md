# Phase 03 — Execution Engine

## Mission

Turn AEGIS from mapped coordination into autonomous artifact production.

## Ships in this phase

- Forge.
- Loop.
- Execution classes.
- Backlog digestion.
- Artifact packaging rules.
- Post-run self-review.

## Banned in this phase

- Broad business expansion without proof.
- Fancy society agents that do not improve execution throughput.

## Acceptance criteria

- Forge can produce one complete artifact from a mapped intent.
- Loop can select, sequence, and finish work from a bounded backlog.
- Outputs are resumable, reviewable, and linked to trace ids.

## Proof demo

One intent becomes a real shipped artifact with tests, docs, packaging, and a stored consequence map.
