# System Map

## Top-level shape

AEGIS is composed of:

- one kernel,
- one standing society,
- one unified session,
- one memory spine,
- one policy layer,
- one observability surface,
- and many possible artifact outputs.

## Flow

1. Human intent arrives.
2. Kernel normalizes the intent into events.
3. Warden validates structure and delegation rights.
4. Scribe loads relevant memory.
5. Herald maintains session continuity across interfaces.
6. Forge and Loop plan and execute.
7. The consequence mapper records every critical step.
8. Outcome, risk, and wealth impact are persisted.

## Design rule

The map must always be simpler than the mythology around it.
