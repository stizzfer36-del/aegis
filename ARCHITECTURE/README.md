# Architecture

This folder defines the contracts that keep AEGIS real.

## What belongs here

- Shared system models.
- Event and trace definitions.
- Delegation logic.
- Memory and policy structures.
- Wealth and observability contracts.

## What does not belong here

- Agent-specific personality text.
- UI-only decisions.
- Temporary experiments without system impact.
